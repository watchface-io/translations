Mostrador rico em dados com personalizações ilimitadas, mais de 100 campos de dados e totalmente otimizado para consumo mínimo de bateria

⭐️ Gratuito vs Pro ⭐️
Este mostrador está disponível sem custos com todas as funções básicas. Características premium (marcadas com [*]) requerem uma Chave Pro para desbloquear. Pode comprar uma Chave Pro aqui: https://ko-fi.com/s/3633613c58 ou https://watchface.io/nexus, ou comprar a versão PRO aqui: https://apps.garmin.com/apps/e3cb3f3d-5587-441e-90c2-43d14118bf3e.

⭐️ Dispositivos MIPS (Fenix 7, Enduro, Forerunner 255, etc.) ⭐️
Este mostrador é totalmente optimizado para MIPS, mas as cores podem parecer diferentes como na maioria das imagens, pois estas são tiradas num dispositivo Amoled. Este mostrador tem muitos temas optimizados para MIPS e suporta gradientes MIPS. Pode encontrar todos os temas integrados aqui: https://watchface.io/nexus/themes#MIPS

⭐️ Configuração ⭐️
Nota: Há dois sítios diferentes com configurações:
* App Garmin IQ (no seu telefone vá para o seu dispositivo, mostradores, configurações): Configure campos de dados, cores e configurações gerais.
* Configurações no relógio: Aceda a opções mais detalhadas como formato de hora, metas e unidades directamente no relógio premindo o botão menu, seleccionando mostrador e tocando personalizar.

⭐️ Campos de dados ⭐️
Todos os campos de dados que a Garmin actualmente fornece para programadores terceiros estão disponíveis, mais extras como suporte a Monitor de Glucose (CGM), Open Weather Map (OWM) e suporte para outras aplicações terceiras.
Todos os campos de dados disponíveis estão listados aqui: https://watchface.io/docs/datafields. Alguns campos de dados (como prontidão para treino ou HRV) actualmente não estão disponíveis para programadores terceiros, veja aqui: https://watchface.io/faq/can-you-add-training-readiness

⭐️ Posso alterar o texto Fenix/Enduro, etc.? ⭐️
Sim, pode! Veja aqui: https://watchface.io/faq/how-to-change-the-fenix-text
Na verdade, tudo neste mostrador pode ser ajustado ao seu gosto.
