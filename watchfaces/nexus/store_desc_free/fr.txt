Cadran riche en données avec personnalisations illimitées, plus de 100 champs de données et entièrement optimisé pour une consommation minimale de batterie

⭐️ Gratuit vs Pro ⭐️
Ce cadran est disponible gratuitement avec toutes les fonctions de base. Les fonctionnalités premium (marquées avec [*]) nécessitent une Clé Pro pour débloquer. Vous pouvez acheter une Clé Pro ici : https://ko-fi.com/s/3633613c58 ou https://watchface.io/nexus, ou acheter la version PRO ici : https://apps.garmin.com/apps/e3cb3f3d-5587-441e-90c2-43d14118bf3e.

⭐️ Appareils MIPS (Fenix 7, Enduro, Forerunner 255, etc.) ⭐️
Ce cadran est entièrement optimisé pour MIPS, mais les couleurs peuvent paraître différentes comme dans la plupart des images, car celles-ci sont prises sur un appareil Amoled. Ce cadran a de nombreux thèmes optimisés pour MIPS et prend en charge les dégradés MIPS. Vous pouvez trouver tous les thèmes intégrés ici : https://watchface.io/nexus/themes#MIPS

⭐️ Configuration ⭐️
Note : Il y a deux endroits différents avec des paramètres :
* App Garmin IQ (sur votre téléphone allez à votre appareil, cadrans, paramètres) : Configurez les champs de données, couleurs et paramètres généraux.
* Paramètres sur la montre : Accédez à des options plus détaillées comme le format de l'heure, les objectifs et les unités directement sur la montre en maintenant le bouton menu, sélectionnant cadran et appuyant sur personnaliser.

⭐️ Champs de données ⭐️
Tous les champs de données que Garmin fournit actuellement aux développeurs tiers sont disponibles, plus des extras comme le support Moniteur de Glucose (CGM), Open Weather Map (OWM) et le support pour d'autres applications tierces.
Tous les champs de données disponibles sont listés ici : https://watchface.io/docs/datafields. Certains champs de données (comme la préparation à l'entraînement ou HRV) ne sont actuellement pas disponibles pour les développeurs tiers, voir ici : https://watchface.io/faq/can-you-add-training-readiness

⭐️ Puis-je changer le texte Fenix/Enduro, etc. ? ⭐️
Oui, vous pouvez ! Voir ici : https://watchface.io/faq/how-to-change-the-fenix-text
En fait, tout sur ce cadran peut être ajusté à votre goût.
