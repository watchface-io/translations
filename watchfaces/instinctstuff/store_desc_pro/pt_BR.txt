Mostrador digital retrô pixel-perfeito. Exibe até 8 métricas de dados esportivos em tempo real com marcadores nítidos, barras de progresso suaves, gradientes sutis e suporte para mais de 100 campos de dados Garmin. Ultra econômico em bateria e totalmente otimizado para todos os dispositivos suportados (incluindo Venu X1 e outras telas quadradas).

⭐️ Dispositivos MIP (Fenix 7, Enduro, Forerunner 255, etc.) ⭐️
As cores podem parecer diferentes das da maioria das imagens, pois são tiradas em um dispositivo Amoled. Este mostrador tem muitos temas otimizados para MIP e suporta gradientes MIP. Você pode encontrar todos os temas integrados aqui: https://watchface.io/quantum_square/themes#MIP

⭐️ Configuração ⭐️
Nota: Existem dois lugares diferentes com configurações:
* App Garmin IQ (no seu telefone vá para seu dispositivo, mostradores, configurações): Configure campos de dados, cores e configurações gerais.
* Configurações no relógio: Acesse opções mais detalhadas como formato de hora, metas e unidades diretamente no relógio mantendo pressionado o botão de menu, selecionando mostrador e tocando em personalizar.

⭐️ Campos de dados ⭐️
Todos os campos de dados que a Garmin atualmente fornece para desenvolvedores terceiros estão disponíveis, além de extras como suporte a Monitor de Glicose (CGM), Open Weather Map (OWM) e suporte para outros aplicativos terceiros.
Todos os campos de dados disponíveis estão listados aqui: https://watchface.io/docs/datafields. Alguns campos de dados (como prontidão para treinamento ou HRV) não estão disponíveis atualmente para desenvolvedores terceiros, veja aqui: https://watchface.io/faq/can-you-add-training-readiness
